## LESSONS.md Rules (Non-Negotiable)

1. **`crate-type = ["cdylib", "rlib"]`** ‚Äî declare both in Cargo.toml so `cargo test` works
2. **PyO3 class names**: Python imports the Rust struct name. If `struct IndexManager` is exported
   via `#[pyclass]`, Python does `from rust_core import IndexManager`. Add `as PyIndexManager`
   alias in Python if needed.
3. **Lock ordering** (deadlock prevention ‚Äî ABSOLUTE RULE):
   - Level 1: atomic state variables (compare_exchange, fetch_add)
   - Level 2: outer `tables: RwLock<HashMap<String, Arc<Table>>>` ‚Äî acquire read/write, clone Arc, RELEASE immediately
   - Level 3: per-collection `graph: Arc<RwLock<KnowledgeGraph>>` ‚Äî held only for actual operation
   - Level 4: leaf utilities (Tantivy IndexWriter, LRU cache Mutex)
   - **Never** hold a Level 2 lock while acquiring Level 3. Never reverse order.
4. **Write semaphore**: `Semaphore::new(1)` for LanceDB batch writes (serialized)
5. **Search semaphore**: `Semaphore::new(100)` for concurrent searches

### 6. File scanner (`src/ingestion/scanner.rs`)

- `FileScanner::scan(&self) -> Vec<FileEntry>` ‚Äî recursive WalkDir with depth/count limits

- Block extensions: `["exe", "sh", "bat", "cmd", "ps1", "py", "rb", "pl", "key", "pem", "p12", "pfx", "env", "sqlite", "db"]`

### 7. Text extractor (`src/ingestion/extractor.rs`)

Trait `TextExtractor` + implementations:
- `PdfExtractor`: use `lopdf` for text extraction; fall back to `pdf-extract`
- `PlainTextExtractor`: direct UTF-8 read
- `MarkdownExtractor`: `pulldown-cmark` ‚Üí strip tags ‚Üí plain text

### 8. Chunker (`src/ingestion/chunker.rs`)

- `Chunker::new(chunk_size: usize, overlap: usize) -> Self`
- `Chunker::chunk_document(doc: &ExtractedDocument) -> Vec<RawChunk>`
- Use `text-splitter` crate with semantic boundary awareness (paragraph breaks first, then token)
- Default: 512-token chunks, 50-token overlap

### 9. WAL writer and recovery (`src/wal/`)

- `WalWriter::append(entry: &WalEntry) -> Result<()>` ‚Äî JSON-line append to file
- `WalEntry`: timestamp, operation (InsertNode/InsertEdge/DeleteEdge), collection_id, payload
- `WalRecovery::replay(db_path: &str) -> Result<()>` ‚Äî read WAL, replay into LanceDB + petgraph
- On startup: replay then truncate WAL (checkpoint)

## Constraints (What NOT to Do)

- Do NOT use PostgreSQL anywhere in Rust code
- Do NOT embed Python model loading in Rust ‚Äî embeddings are passed as `Vec<f32>` from Python
- Do NOT hold Level 2 lock while doing any I/O or acquiring Level 3 lock

  or use `block_on` with a thread-local runtime
- Do NOT add `#[pyclass(name = "PyIndexManager")]` unless Python code explicitly uses that name
